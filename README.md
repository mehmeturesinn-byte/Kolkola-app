# KolKola - Refakatçi İsteme Uygulaması 🤝

## Proje Açıklaması

**KolKola**, yalnız olanlar ve destek ihtiyacı duyanlar ile profesyonel refakatçıları bir araya getiren bir platformdur.

## Özellikler

- 👤 Kullanıcı ve refakatçı kayıt sistemi
- 📅 Randevu oluşturma ve yönetimi
- 💳 Ücretli hizmet alma
- 🗺️ Konum tabanlı refakatçı arama
- 💬 Gerçek zamanlı sohbet
- ⭐ Değerlendirme sistemi
- 📱 iOS ve Android mobil uygulaması

## Tech Stack

- Backend: Node.js + Express + TypeScript
- Frontend: React + TypeScript
- Mobile: React Native (Expo)
- Database: PostgreSQL

## Proje Yapısı

```
kolkola-app/
├── backend/
├── frontend/
├── mobile/
└── docs/
```

## Başlangıç

### Kurulum

```bash
cd mobile
npm install
npm start
```
