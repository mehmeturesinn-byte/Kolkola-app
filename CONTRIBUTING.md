# Katkı Rehberi

Katkı veren herkese teşekkür ederiz! 🙏

## Issue Bildirme

1. GitHub Issues'e gidin
2. Açık bir başlık yazın
3. Detaylı açıklama ekleyin

## Pull Request

1. Fork edin
2. Feature branch oluşturun: `git checkout -b feature/your-feature`
3. Commit atın: `git commit -m "Add: feature description"`
4. Push edin: `git push origin feature/your-feature`
5. Pull Request oluşturun
